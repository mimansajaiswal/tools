        Modal Embed Variant

        Simulates usage inside a constrained modal container (scrolling body, clipped parent, fixed overlay). Each open starts from one random preset quick-add input (editable). Click Save to append rows (add-only) from getResult() into the table preview. Defaults include file refs and matching seeded global demo files so previews are visible immediately. Dropdowns and calendar pickers are positioned within modal/container bounds.

        The modal body is intentionally constrained to mimic embedded app layouts.
